# DigiTwin
Détection précoce du sepsis grâce au jumeau numérique

**## Le projet**
DigiTwin est un jumeau numérique médical fondé sur l'intelligence artificielle, destiné à la détection précoce du sepsis. Le modèle prédit le risque de sepsis six heures à l'avance en combinant données cliniques réelles et contraintes physiologiques.

**## Fonctionnalités**
- Prédiction du risque de sepsis à +6 heures
- Plateforme de monitoring en temps réel
- Rapport explicable automatique (SHAP, LIME)
- Développé en collaboration avec des médecins

**## Stack technique**
- Python
- TensorFlow / PyTorch
- Grafana / Inluxdb
- SHAP / LIME
